# 🧠 מבחן אינטראקטיבי בטלגרם

## ✨ פיצר חדש: "בחן אותי בטלגרם"

הבוט תומך עכשיו במבחנים אינטראקטיביים ישירות בטלגרם! 

## 🎯 איך זה עובד?

### 1. יצירת מבחן רגיל
- העלה קובץ כרגיל (PDF, DOCX, TXT)
- בחר מספר שאלות (3-50)
- המתן שהמבחן יוכן

### 2. בחירת מבחן אינטראקטיבי
אחרי שהמבחן מוכן, תראה כפתור חדש:
```
🧠 בחן אותי בטלגרם
```

### 3. חוויית המבחן
- **שאלות אחת אחת** עם כפתורי בחירה
- **מעקב התקדמות** (שאלה 3/10)
- **קושי מוצג** (🟢 קל, 🟡 בינוני, 🔴 קשה, ⚫ מאוד קשה)
- **תשובה מיידית** אחרי כל שאלה עם הסבר

### 4. תוצאות סופיות
בסיום המבחן תקבל:
- **ציון באחוזים** 🎯
- **זמן שלקח** ⏱️
- **פילוח לפי קושי** 📈
- **דירוג ביצועים** (🥇🥈🥉👍📚)

## 📋 תכונות מתקדמות

### הגבלות חכמות
- **מקסימום 10 שאלות** במבחן אינטראקטיבי (לחוויה אופטימלית)
- **ערבוב אוטומטי** של השאלות הנבחרות
- **ניקוי אוטומטי** של מבחנים ישנים (2 שעות)

### חישוב ציונים
| ציון | דירוג | אמוג'י |
|------|--------|--------|
| 90%+ | מעולה! | 🥇 |
| 80-89% | טוב מאוד! | 🥈 |
| 70-79% | טוב! | 🥉 |
| 60-69% | עובר | 👍 |
| <60% | צריך שיפור | 📚 |

### אחרי המבחן
כפתורים זמינים בסיום:
- **🧠 בחן אותי שוב** - מבחן חדש מאותו חומר
- **🔄 מבחן נוסף** - מבחן HTML עם מספר שאלות שונה  
- **📄 הורד מבחן HTML** - גרסה מלאה להורדה
- **🆕 מבחן חדש** - התחל עם חומר חדש
